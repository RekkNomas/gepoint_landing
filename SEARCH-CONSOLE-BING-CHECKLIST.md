# Search Console + Bing Webmaster Checklist

## 1) Verificación de propiedad
- [ ] Agregar propiedad de dominio en Google Search Console: `gpsolution.es`
- [ ] Agregar propiedad en Bing Webmaster Tools
- [ ] Verificar por DNS (TXT) para cobertura total de subdominios/protocolos

## 2) Sitemaps
- [ ] Enviar sitemap en Google: `https://www.gpsolution.es/sitemap-index.xml`
- [ ] Enviar sitemap en Bing: `https://www.gpsolution.es/sitemap-index.xml`
- [ ] Confirmar estado `Success` / `Processed`

## 3) Indexación inicial
- [ ] Inspeccionar URL home: `https://www.gpsolution.es/`
- [ ] Solicitar indexación de:
  - `/`
  - `/aviso-legal`
  - `/privacidad`
  - `/cookies`

## 4) Cobertura y rastreo
- [ ] Revisar páginas excluidas por `noindex` accidental
- [ ] Revisar errores 404/5xx
- [ ] Verificar que `robots.txt` no bloquee páginas clave

## 5) Calidad SEO
- [ ] Revisar Core Web Vitals (LCP, INP, CLS)
- [ ] Validar Rich Results (Organization, WebSite, FAQPage, BreadcrumbList)
- [ ] Confirmar canonical correcto en páginas clave

## 6) Seguimiento mensual
- [ ] Exportar consultas + CTR + posiciones top
- [ ] Detectar páginas con impresiones y bajo CTR (optimizar titles/descriptions)
- [ ] Detectar keywords emergentes y crear nuevas páginas/FAQs
